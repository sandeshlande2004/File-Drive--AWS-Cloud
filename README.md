# File-Drive--AWS-Cloud
Developed a cloud based file uploading system using MERN stack and AWS S3 bucket. 
1. Frontend and Backend developed using React Js and Node Js.
2. Used Express Js to integrate Frontend and Backend.
3. AWS S3 bucket to store uploaded files.
4. Mongodb to store Metadata of the files.

File Drive System:
i. Options like Upload, Delete and View
ii. Can access through any place using username and password.
iii. Interactive and simple user interface in React Js.
iv. Easy to use and understand
